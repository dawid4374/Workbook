### Usage

Install packages:
 ```
 $ npm install
 
 ```
Run app:
 ```
$ npm start
# Or run with Nodemon
$ npm run dev

# Visit http://localhost:3000
```

### MongoDB 

Open "config/config.env" and add your MongoDB URI, local or Atlas
